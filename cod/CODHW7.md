### 3.

1. 海明码应为7位，不妨设三位校验码从左往右依次为A1, A2, A4

   ```
   1100 -> A1 A2 1 A4 1 0 0	元数据分别编号为A3, A5, A6, A7
   A1 = A3 ⊕ A5 ⊕ A7 = 1 ⊕ 1 ⊕ 0 = 0
   A2 = A3 ⊕ A6 ⊕ A7 = 1 ⊕ 0 ⊕ 0 = 1
   A4 = A5 ⊕ A6 ⊕ H7 = 1 ⊕ 0 ⊕ 0 = 1
   1100 -> 0111100
   同理
   1101 -> 1010101
   1110 -> 0010110
   1111 -> 1111111
   ```

2. 设置三位S1, S2, S3用于校验（从左往右为低位至高位）

   ```
   1100100:
   S1 = A1 ⊕ A3 ⊕ A5 ⊕ A7 = 1 ⊕ 0 ⊕ 1 ⊕ 0 = 0
   S2 = A2 ⊕ A3 ⊕ A6 ⊕ A7 = 1 ⊕ 0 ⊕ 0 ⊕ 0 = 1
   S3 = A4 ⊕ A5 ⊕ A6 ⊕ A7 = 0 ⊕ 1 ⊕ 0 ⊕ 0 = 1
   即A6出错，取反即可
   1100111:
   S1 = 1 ⊕ 0 ⊕ 1 ⊕ 1 = 1
   S2 = 1 ⊕ 0 ⊕ 1 ⊕ 1 = 1
   S3 = 0 ⊕ 1 ⊕ 1 ⊕ 1 = 1
   即A7出错，取反即可
   1100000:
   S1 = 1 ⊕ 0 ⊕ 0 ⊕ 0 = 1
   S2 = 1 ⊕ 0 ⊕ 0 ⊕ 0 = 1
   S3 = 0 ⊕ 0 ⊕ 0 ⊕ 0 = 0
   即A3出错，取反即可
   1100001:
   S1 = 1 ⊕ 0 ⊕ 0 ⊕ 1 = 0
   S2 = 1 ⊕ 0 ⊕ 0 ⊕ 1 = 0
   S3 = 0 ⊕ 0 ⊕ 0 ⊕ 1 = 1
   即A4出错，取反即可
   ```
